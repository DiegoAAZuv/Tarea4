# Comparador de 4 bits
Simulación de un comparador binario de 4 bits para la asignatura Hardware Digital.

## Descripción
El sistema recibe dos valores binarios de 4 bits y determina su relación de magnitud mediante tres salidas: mayor que, igual a y menor que. El diseño fue realizado para simularse en Tinkercad con interruptores como entradas y LEDs como indicadores de salida.
Este diseño fue elegido debido a su simpleza de desarrollo y su correlacion con la materia vista con anterioridad en la asignatura, sirvio para desarrollar un mejor entendimiento sobre las compuertas 

## Archivos del repositorio
- `comparador_4_bits.json`: simulación del circuito.
- `README.md`: descripción general del proyecto.
- `Tarea4_Individual_ISF215.pdf`: enunciado de la actividad.

## Casos de prueba
estos casos fueron pruebas realizadas previo a la simulacion grabada, utilizadas para corroborar el correcto funcionamiento del codigo y la construccion del sistema.
- `0101` vs `0101` → igualdad.
- `1010` vs `0111` → mayor.
- `0011` vs `1000` → menor.
